## Batch 4 — Consistency Polish (45 min, low risk)

### 12. Standardize button border radius
**Target standard:**
| Element | Radius |
|---------|--------|
| Badges/pills | `rounded-full` |
| Inputs, nav items, small buttons | `rounded-lg` |
| Primary CTA buttons | `rounded-xl` |
| Cards | `rounded-xl` |
| Feature cards/modals | `rounded-2xl` |

**Key files to update:**
- `components/site-header.tsx` — nav items `rounded-sm` → `rounded-lg`
- `components/mobile-services-accordion.tsx` — `rounded-sm` → `rounded-lg`
- `components/cta-section.tsx` — buttons `rounded-sm` → `rounded-xl`
- `app/th/page.tsx` — "View All Services" `rounded-full` → `rounded-xl`
- `app/en/page.tsx` — "View All Services" `rounded-lg` → `rounded-xl`
- `components/about-preview-section.tsx` — keep `rounded-xl`
- `components/free-consultation-section.tsx` — `rounded-2xl` → `rounded-xl`
